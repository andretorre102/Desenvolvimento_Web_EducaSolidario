# EducaSolidário

## Sobre o Projeto

Plataforma web para conectar doadores a instituições de ensino (escolas, ONGs e creches) que precisam de apoio financeiro. O site apresenta as instituições em destaque, mostra estatísticas de arrecadação e permite que o visitante conheça o perfil de cada instituição para realizar doações.

**Principais funcionalidades:**

- Página inicial com estatísticas gerais (instituições cadastradas, valor doado, doadores ativos)
- Listagem de instituições em destaque, organizadas por categoria (Escola, ONG, Creche)
- Exibição do progresso de arrecadação de cada instituição (percentual da meta atingida)
- Botões de ação para visualizar o perfil da instituição ou realizar uma doação
- Layout responsivo, adaptado para desktop e dispositivos móveis
- Cadastro e login de usuários (em desenvolvimento)

## Como Usar

### Pré-requisitos

- Navegador web atualizado (Chrome, Firefox, Edge ou similar)
- Não é necessário nenhum servidor ou instalação adicional, pois o projeto é estático (HTML, CSS e JavaScript)

### Instalação

```bash
git clone https://github.com/usuario/EducaSolidario.git
cd EducaSolidario
```

### Executando o Projeto

Basta abrir o arquivo `index.html` diretamente no navegador, ou utilizar uma extensão como o **Live Server** (VS Code) para servir os arquivos localmente.
